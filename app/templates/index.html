Secure File Sharing API

API Documentation

This is a RESTful API for secure file sharing between Operations and Client users.

Authentication

  • POST /auth/signup - Register a new client user
  • GET /auth/verify/{token} - Verify email address
  • POST /auth/login - Login for both Ops and Client users

Files

  • POST /file/upload - Upload files (Ops users only)
  • GET /file/list - List all files (Client users only)
  • GET /file/download/{file_id} - Get download link (Client users only)
  • GET /file/download-file/{token} - Download file using token (Client users only)
